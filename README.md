# UserAuthentication
A webpage to authenticate user after creating accounts.

### feature:
* Sign-up
* Registration
* Sign out
* Reset Password
* Password stored in encrypted form
* Login using Google(social authentication)
* Forgot password
* Notication for succes and login
* Recaptcha in login

## Tools Used:
* HTML
* CSS
* Ajax
* JavaScript
* Nodejs
* Express
* MongoDb
* Passport
* Flash
* Google passport strategy
